# Superstore Sales Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to uncover insights about sales performance, customer segments, and profitability.

The goal of this project is to demonstrate **data analysis, visualization, and business insight generation using Python**.

---

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook (VS Code)

---

## Dataset

The dataset contains transactional sales data including:

* Order information
* Customer details
* Product categories
* Sales and profit metrics

Key columns include:

* Sales
* Profit
* Discount
* Quantity
* Category
* Region
* City
* Order Date

## Data Cleaning

The following preprocessing steps were performed:

* Checked for missing values
* Converted date columns to datetime format
* Verified dataset structure
* Ensured correct data types

Result:
The dataset contained **no missing values** and was ready for analysis.

## Exploratory Data Analysis

### 1. Monthly Sales Trend

Shows how sales change over time to identify seasonal patterns.

### 2. Sales by Region

Compares revenue across geographic regions.

### 3. Top 10 Cities by Sales

Identifies cities generating the highest revenue.

### 4. Sales by Category

Analyzes which product categories generate the most sales.

### 5. Discount vs Profit Analysis

Examines how discount levels impact profitability.

### 6. Correlation Heatmap

Visualizes relationships between sales, profit, discount, and quantity.

---

## Key Business Insights

* A small number of cities generate a large portion of revenue.
* High discount levels can negatively affect profitability.
* Some product categories contribute significantly more to total sales.
* Sales show trends over time that may indicate seasonal demand.
